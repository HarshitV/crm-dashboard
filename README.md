# CRM Dashboard

A simple CRM dashboard built with React and TypeScript.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Run Storybook:
   ```sh
   npm run storybook
   ```

## Project Structure

- `src/` - Main source code
- `public/` - Static assets
